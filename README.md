# Sketchware IDE

Sketchware IDE is a modified version of Sketchware Pro, redesigned to be a more flexible and powerful Integrated Development Environment (IDE) for Android development.

## Key Changes

- **Branding Reset:** All references to "Sketchware Pro" and the original "Besome" developers have been removed or updated to "Sketchware IDE".
- **Free-form Java Coding:** The traditional block-based event system has been bypassed in favor of a direct Java Code Editor. This allows for more complex logic and removes the limitations of the block system.
- **IDE Focus:** Transitioning from a simplified "maker" tool to a professional-grade mobile IDE.

## Features

- **Java Event Editor:** Edit your logic directly in Java using a powerful code editor with syntax highlighting.
- **XML Layout Editor:** Full control over your layouts with the built-in XML editor.
- **Offline Capability:** Designed to work entirely without an internet connection.
- **AI Integration (Planned):** Support for local LLMs (Large Language Models) to provide intelligent code assistance offline.

## AI Integration Research

For local AI functionality on Android, consider the following libraries:
- **MediaPipe LLM Inference API:** Google's official solution for running LLMs on-device.
- **MLC LLM:** A universal deployment solution that allows LLMs to run efficiently on mobile hardware.
- **TensorFlow Lite:** For smaller, custom-trained models.
